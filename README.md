## Installation

> Install with yarn or npm: `yarn` or `npm`:

```bash
# yarn
yarn add nest-keycloak-middleware
```

```bash
# npm
npm i nest-keycloak-middleware --save
```

### Import the lib with es6 or cjs:

```mjs
// es6
import { KeycloakModule } from 'nest-keycloak-middleware';
```
